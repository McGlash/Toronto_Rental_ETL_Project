ETL Project

In this project we extract transform and load the following Toronto datasets: Cragslist Rentals Data, Kijiji Rentals Data, Toronto Police Services Open Data and Community Assets Data

Extract

API calls and web scrapping are used to collect all the data from the different data sources. This data is stored for further manipulation.

Transform

Data is messy !!! Especialy when it comes from four different data sources. Transformations have to be applied to make records compatible with each other. Transformations are also applied to ensure correctness and create relationships between different tables.

Load

PostgreSQL was used to load and store this data. In the case of an SQL Database a Database schema is created first and then the data is loaded. If the data being loaded does not follow the schema there will be errors ....
